## Current Task

### 1. Search for a product using full name and validate results

**Example search query:** `Apple iPhone 14`

**Expected behavior:**
- Search the product using the full name
- Verify that the correct product appears in the results
- Validate that the result matches the expected product details

**Validation may include:**
- Product title
- Brand name
- Category
- Price
- Availability
- Exact or partial match checks
